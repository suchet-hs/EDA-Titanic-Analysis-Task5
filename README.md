# 📊 Task 5: Exploratory Data Analysis (EDA)

## 📌 Objective
To extract meaningful insights from data using statistical and visual analysis techniques.

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
Dataset used: Titanic Dataset

---

## 🔍 Key Concepts Covered
- Data Cleaning and Exploration
- Statistical Summary (.describe(), .info())
- Data Visualization
- Correlation Analysis
- Pattern and Trend Identification

---

## 📊 Analysis Performed

### 1. Data Overview
- Checked dataset structure using `.info()` and `.describe()`

### 2. Data Cleaning
- Handled missing values
- Checked duplicates

### 3. Univariate Analysis
- Used histograms and count plots

### 4. Bivariate Analysis
- Compared survival with gender, class, and age

### 5. Multivariate Analysis
- Used heatmap and pairplot to understand relationships

### 6. Visualization
- Histograms
- Boxplots
- Scatter plots
- Heatmap

---

## 📸 Output
All analysis is available in the Jupyter Notebook.

---

## 🎯 Key Insights
- Survival rate varies based on gender and class
- Higher class passengers had higher survival rates
- Age and fare show certain patterns

- Learned how to explore and visualize data
